# PSX Investor Toolkit

This Python application is a powerful finance tool for investors in Pakistan. It helps investors perform financial analysis for stocks listed on the Pakistan Stock Exchange, compute intrinsic values, get current stock prices, and perform tax calculations.

**Application Description**
This application is developed using Gradio for the front end and Yahoo Finance for financial market information.

**Description**
This application is a simple financial terminal application that combines investment and tax tools for investors.

**🚀 Features**
**📈 PSX Stock Search**

Search any company listed on the Pakistan Stock Exchange

Fetch available tickers dynamically using Yahoo Finance

Automate the process of extracting tickers from search results

No need to manually enter tickers

**📊 Stock Valuation Tool**

Estimate the intrinsic value of a company using multiple financial models

Features include:

Retrieve current stock prices

Estimate Cost of Equity using the CAPM Model

Estimate Weighted Average Cost of Capital using the WACC Model

Estimate Earnings Growth

Estimate Future Earnings

**Estimate Fair Values**

Estimate intrinsic values using the following financial models:

**P/E Valuation Method**

**Dividend Discount Model (DDM)**

**Investment Decision Metrics**

Margin of Safety (MOS)
💹 Stock Price Checker

Get the current market price of any listed company on the PSX instantly.

Features:

Dynamic search for companies

Instant retrieval of stock price

Displays price instantly

💰 Income Tax Calculator

Calculate the annual income tax in Pakistan.

Supports:

Salaried Persons

Business Income

Features:

Calculations based on progressive tax slabs

Automatic calculation of tax

Additional surcharge for high-income earners

📉 Capital Gain Tax Calculator

Calculate the tax payable on capital gains depending on the holding period.

Features:

Calculates the tax payable on capital gains

Supports multiple holding periods

Displays:

Tax Payable

Net Gain After Tax

🌐 Application Interface

The application toolkit comprises the following modules:

Stock Valuation Tool

Stock Price Checker

Income Tax Calculator

Capital Gain Tax Calculator
